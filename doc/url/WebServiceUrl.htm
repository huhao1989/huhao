文件上传服务. 包括企业文件和个人文件上传

请求地址:"/sc/file/fileRoutingUpload"

参数:

  • FileUploadDTO 文件上传DTO

正常返回:

  • error500 服务器内部错误
  • errorSameFile 文件重名
  • errorNoSpace 个人网盘空间不足

异常返回:

  • error500 服务器内部错误
  • errorSameFile 文件重名
  • errorNoSpace 个人网盘空间不足
  • errorNoPermission：用户没有当前文件夹下的上传权限
  • errorFolderSpaceOver：当前文件夹空间已满
  • errorNoSpace：企业网盘空间已满
  • errorSameFile：同一文件夹下存在同名文件
  • errorFolderDeleted：父文件夹已经删除
  • error500：系统错误

浏览图片，取图片尺寸及地址

请求地址:"/sc/file/viewFileAsImage"

参数:

  • ViewFileDTO的JSON：文件转换浏览信息

正常返回:

  • ViewFileResultDTO.message = CommConstants.OK_MARK： 转换成功，ViewFileResultDTO.url为图片地址，ViewFileResultDTO.width 为图片宽度px，ViewFileResultDTO.height为图片高度px
  • ViewFileResultDTO.message = errorNoPermission：没有浏览权限
  • ViewFileResultDTO.message = errorNotSupported：该文件不支持图片预览
  • ViewFileResultDTO.message = errorFileNotFound：文件不存在
  • ViewFileResultDTO.message = error500：系统错误

异常返回:

  • error500：系统错误
